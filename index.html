Persona Architect

A lightweight CLI-based AI assistant framework that supports memory, session saving, persona-driven response generation, and OpenAI integration (or offline stub mode).

🎯 Features

  • CLI-based interface for interacting with your AI persona
  • Persona system with customizable tone, goals, and constraints
  • Session logging (stored with timestamped filenames)
  • Offline stub responses or real OpenAI integration
  • Modular design (easy to extend or plug into other tools)

📦 Project Structure

  • main.py – Main entrypoint (interactive chat loop)
  • memory.py – Tracks multi-turn dialogue context
  • session_saver.py – Logs conversations with timestamps
  • prompt_builder.py – Constructs persona-aware prompts
  • response_handler.py – Stub/OpenAI response engine

🧠 Memory + Session Logging

All dialogue is tracked and saved when the user exits via exit or quit. Session logs are timestamped and saved locally.

🔧 To Run

python main.py

🔐 OpenAI Integration (Optional)

To enable OpenAI response logic, set USE_OPENAI = True and insert your API key in config.py.
